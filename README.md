# CotizAI 📄

Generador de cotizaciones profesionales en PDF — funciona como app instalable en Android e iOS.

**Demo:** https://ChristianSPP.github.io/cotizai-app

---

## 🚀 Cómo subir a GitHub Pages

### 1. Crea el repositorio
- Ve a github.com → New repository
- Nombre: `cotizai-app`
- Visibilidad: **Public**
- Clic en Create repository

### 2. Sube el archivo
- Clic en "uploading an existing file"
- Arrastra `index.html`
- Clic en Commit changes

### 3. Activa GitHub Pages
- Settings → Pages
- Branch: **main** → Save
- En 1-2 minutos estará en línea ✅

---

## 📱 Instalar en Android

1. Abre Chrome en tu celular
2. Entra a `https://TU-USUARIO.github.io/cotizai-app`
3. Aparece un banner azul → toca **"Instalar"**
4. ¡Listo! Queda en tu pantalla de inicio como app

## 🍎 Instalar en iPhone/iPad

1. Abre Safari (debe ser Safari, no Chrome)
2. Entra a tu URL
3. Toca el botón compartir (□↑)
4. Toca **"Añadir a pantalla de inicio"**

## ✨ Características

- ✅ Genera PDFs profesionales con logo y colores de marca
- ✅ Pagos únicos y mensuales en la misma cotización
- ✅ IVA configurable (incluido o excluido)
- ✅ RUT opcional
- ✅ Disclaimer / aviso legal automático
- ✅ Sugerencia de ítems con IA
- ✅ 6 paletas de color + colores personalizados
- ✅ Funciona sin internet una vez instalada (PWA)
- ✅ Sin servidor, sin base de datos, 100% en el navegador
